# Synthetic Data Pack

This directory contains deterministic sample datasets created exclusively for portfolio use. Every file is fictitious and safe to share publicly.

## Included Files

- `dim_partner.csv`: gym partner dimension used for partner performance analysis
- `dim_member.csv`: synthetic member records with partner, signup, and status attributes
- `fact_membership.csv`: membership plan and fee information for lifecycle analysis
- `fact_checkin.csv`: event-level gym check-ins for engagement and recency metrics
- `fact_class_booking.csv`: class booking outcomes for attendance behavior analysis
- `membership_export_raw.csv`: intentionally messy extract designed for the Python cleaning example
- `membership_export_clean.csv`: cleaned export generated by the pandas transformation script
- `dim_account.csv`: SaaS account dimension for recurring revenue analysis
- `dim_plan.csv`: SaaS plan dimension with tier labels
- `fact_subscription_snapshot.csv`: monthly recurring revenue snapshots by account
- `it_service_tickets.csv`: synthetic service operations dataset for SLA and efficiency reporting

## How The Data Was Built

- The generator script lives in `Python_Automation/generate_synthetic_portfolio_data.py`
- Output is deterministic, which means the same seed always recreates the same files
- Dates, names, accounts, and business events were designed to look realistic without matching any real entity

## Suggested Usage

- Load the gym tables into PostgreSQL, DuckDB, or Databricks to test the SQL challenges
- Use `membership_export_raw.csv` with `Python_Automation/clean_membership_data.py`
- Use `it_service_tickets.csv` as a dashboard prototyping source for BI visuals
- Install dependencies from `requirements.txt` before running the pandas cleaning workflow
